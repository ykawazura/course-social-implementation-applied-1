    5. グループ課題

    課題の目的と進め方

    各グループでパラメータを系統的に変化させ、結果を比較・考察します。 「変えるのは1つのパラメータだけ」が科学的比較の基本です。

    進め方：

    1. まずテーマ A〜D をすべて順番に試す（各テーマで指定されたパラメータ値でシミュレーションを実行）
    2. スクリーンショットや Nu の値を記録する
    3. 時間に余裕があれば、複数のパラメータを同時に変化させた組み合わせ実験にも挑戦する
      • 例：Ra を高くしつつ Pr も変える、高解像度で Ra を変えるなど
    4. 結果をまとめ考察し、レポート作成

    テーマ A：レイリー数（Ra）の効果

    変更するパラメータ： Ra（他はすべてデフォルト値）

    条件 Ra の値 予想される現象
    低 Ra 1000 対流なし（Ra < 1708） → 純伝導状態
    標準 10000 対流あり（デフォルト）
    高 Ra 100000 強い対流 → 複雑な流れ構造（dt = 2e-5 くらいまで下げないと計算が破綻する）

    観察ポイント：

    • 温度場は縞模様になるか？
    • 渦度の最大値（max|ω|）はどう変わるか？
    • ヌセルト数 Nu はどう変化するか？

    考察ヒント：

    • Ra < 1708 では対流が起きないのはなぜか？（浮力 vs 粘性・拡散）
    • Ra が10倍になると Nu は何倍になるか？（スケーリング則：\(Nu \propto Ra^{0.3}\) 程度）

    テーマ B：プラントル数（Pr）の効果

    変更するパラメータ： Pr（他はすべてデフォルト値）

    条件 Pr の値 対応する流体
    低 Pr 0.01 液体金属（水銀など）
    標準 0.71 空気（デフォルト）
    高 Pr 7 水 （dt = 5e-6 くらいまで下げないと計算が破綻する）

    観察ポイント：

    • 温度場と渦度場の広がり方はどう違うか？
    • 対流セルの数や形はどう変わるか？

    考察ヒント：

    • Pr が小さい（液体金属）と、熱が運動量より速く伝わる。温度場は滑らかになる？激しくなる？
    • Pr が大きい（水）と、粘性の効果が強まる。対流セルの形はどう変わる？

    テーマ C：グリッド解像度の効果

    変更するパラメータ： Nx, Ny（他はすべてデフォルト値）

    条件 Nx Ny 格子数（合計）
    粗い 64 32 約 2,000 点
    標準 128 64 約 8,000 点（デフォルト）
    細かい 256 128 約 33,000 点 （dt = 1e-5 くらいまで下げないと計算が破綻する）

    観察ポイント：

    • 解像度が粗いと流れ構造はどう変わるか？
    • 解像度を細かくすると計算結果は変わるか？（収束性の確認）

    考察ヒント：

    • 格子が粗いと「数値拡散」が増える——流れが実際より滑らかに見える
    • 格子を2倍細かくすると計算時間は約4〜8倍になる（格子数が4倍、安定条件で dt も小さくなる場合がある）

    計算時間に注意

    Nx=256, Ny=128 の細かいグリッドでは、Google Colab の無料プランで n_steps=10000 を実行すると 10 分以上かかる場合があります。 先に n_steps=3000 程度で試してから、必要に応じて増やしてください。

    テーマ D：アスペクト比（Lx）の効果

    変更するパラメータ： Lx（他はすべてデフォルト値）

    条件 Lx の値 ドメインの形 予想されるセル数
    正方形 1.0 縦長 1〜2 個
    標準 2.0 2:1（デフォルト） 2 個
    幅広 4.0 4:1 4 個

    観察ポイント：

    • ドメインの幅が広くなるほど対流セルの数はどう変わるか？
    • セルの形（アスペクト比）は揃っているか？

    考察ヒント：

    • ベナールセルの典型的な幅は層の高さとほぼ等しい（アスペクト比 ≈ 1）
    • ドメインが狭い（Lx=1.0）と、セルが1つに収まり切れずに非対称になることがある

    グリッドを大きくする際の注意

    格子数を増やすと計算時間が大幅に増えます。 授業時間内に終わらせるには n_steps を減らすか、Nx/Ny を下げてください。 結果の傾向を掴むことが目的なので、完全な定常状態に達していなくても構いません。
