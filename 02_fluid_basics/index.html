    2. 流体力学の基礎

    流体とは

    流体とは、力を加えると形が変わり続けるもの——つまり気体と液体の総称です。 固体は力を加えてもある程度形を保ちますが、流体は容器の形に合わせて変形します。

    空気・水・油・マントル（地球内部の岩石）はすべて流体として扱えます。

    場（フィールド）という考え方

    流体の状態を記述するには、空間の各点で値が定まる「場」を使います。

    場の名前 記号 意味
    速度場 \(u\), \(v\) 各点の流体がどちら向きに、どのくらいの速さで動いているか
    温度場 \(T\) 各点の温度（下壁=高温、上壁=低温）
    渦度場 \(\omega\) 各点での流体の「回転の強さ」

    たとえば天気予報の「風速マップ」は速度場の可視化、「気温マップ」は温度場の可視化です。 これらの場が時間・空間的にどのように変化していくかを記述するのが流体力学です。

    レイリー数

    レイリー数（Rayleigh number）は、温度差が引き起こす浮力の強さと、 粘性・熱拡散による安定化の比を表す無次元数です。

    • Ra が小さい → 熱は伝導だけで運ばれ、流体は動かない
    • Ra が大きい → 浮力が粘性に打ち勝ち、対流が発生する

    プラントル数

    プラントル数（Prandtl number）は、運動量の拡散と熱の拡散の速さの比です。

    • \(Pr \ll 1\)（液体金属など）: 熱が運動量より速く伝わる
    • \(Pr \approx 1\)（空気）: 熱と運動量が同程度の速さで拡散
    • \(Pr \gg 1\)（水・油・マントル）: 運動量が熱より速く拡散

    支配方程式（無次元形）

    本シミュレーションで解く方程式を示します。 読めなくて構いません——どんな「量」がどんな「式」で決まるかをざっくり眺めてください。

    渦度方程式（流れの回転を記述）：

    \[\frac{\partial \omega}{\partial t} + u\frac{\partial \omega}{\partial x} + v\frac{\partial \omega}{\partial y} = \mathrm{Pr}\,\nabla^2 \omega + \mathrm{Ra}\,\mathrm{Pr}\,\frac{\partial T}{\partial x}\]

    温度方程式（熱の移流と拡散を記述）：

    \[\frac{\partial T}{\partial t} + u\frac{\partial T}{\partial x} + v\frac{\partial T}{\partial y} = \nabla^2 T\]

    流れ関数のポアソン方程式（圧力の代わりに使う補助量 \(\psi\) を求める）：

    \[\nabla^2 \psi = -\omega\]

    速度と流れ関数の関係：

    \[u = \frac{\partial \psi}{\partial y}, \qquad v = -\frac{\partial \psi}{\partial x}\]

    注記： 式の形は気にせず OK です。重要なのは「Ra と Pr というパラメータが流れの性質を決める」という点です。

    鉛直熱フラックスとヌセルト数

    対流がどれだけ効率よく熱を運んでいるかを測る指標がヌセルト数（Nusselt number）です。

    鉛直熱フラックス \(\langle vT \rangle\)

    鉛直速度 \(v\) と温度 \(T\) の積を空間平均した量 \(\langle vT \rangle\) は、対流が運ぶ熱の量を表します。

    • \(\langle vT \rangle = 0\)：対流なし（流体が静止している）
    • \(\langle vT \rangle > 0\)：温かい流体が上昇し、冷たい流体が下降している（対流による熱輸送）

    ヌセルト数 Nu

    壁面では非滑り条件により流速がゼロなので、熱は伝導のみで壁を通過します。 そのため、壁面での実際の熱フラックスはフーリエの法則で決まります：

    \[q = -k \left.\frac{\partial T}{\partial y}\right|_\text{wall}\]

    対流がどれだけ内部で熱を運んでいようと、最終的にすべての熱はこの壁面の温度勾配を通過しなければなりません。 ヌセルト数はこの熱フラックスを純伝導の場合と比較した無次元数です：

    \[Nu = \frac{\text{実際の熱フラックス}}{\text{純伝導の熱フラックス}} = \left\langle -\frac{\partial T}{\partial y}\right\rangle_\text{wall}\]
    • Nu = 1：対流なし、熱は伝導のみで運ばれる
    • Nu > 1：対流が内部をかき混ぜ、壁面付近の温度勾配を急峻にする → 熱輸送が増加

    直感的なイメージ： 鍋をかき混ぜると底からの熱が早く全体に伝わる——その「早さ」の比がNu数です。

    対流が起きる臨界条件

    レイリー数が臨界値 Ra ≈ 1708 を下回る場合、浮力より粘性・熱拡散の効果が勝り、 流体は動かず熱は伝導のみで運ばれます（Nu = 1）。

    Ra > 1708 を超えて初めて対流セルが自発的に形成されます。 ハンズオンでは Ra = 10000 を標準設定としているため、明確な対流が観察できます。
